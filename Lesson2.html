HTML Lists

We have three types of lists, i.e ordered, unordered and description lists

Ordered List

We use the tag ol to create ordered list

Each list item should be enclosed with li tag

Example;
  1. Pray
  2. Warm water
  3. Shower
  4. Breakfast
  5. School

Unordered List

We use the ul tag to create unordered list

Example;
  • Tissue
  • Toothpaste
  • Rollon
  • Pens
  • Soap

Description list

Used to create terms and their meaning

We use the dl tag to create the list, dt for the term and dd for the definition

Milk
White cold drink
Coffee
Black cold drink
